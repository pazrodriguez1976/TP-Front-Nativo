# TP FRONT MEJORADO 

Este repositorio contiene la entrega del Trabajo Práctico de maquetación web. Es una página estática simple construida para demostrar el uso de etiquetas estructurales y estilos básicos.

##  Descripción del TP

La página consta de:
* Un encabezado (`<header>`) con el título principal.
* Una sección principal (`<main>`) que funciona como una galería de imágenes.
* Un pie de página (`<footer>`) con información de cierre.

El diseño incluye un esquema de colores oscuros, bordes redondeados y detalles de delineado en color rojo para resaltar el footer.

##  Cómo visualizar el TP

Como es un proyecto estático sin dependencias:
1. Clonar este repositorio o descargar el archivo `.zip`.
2. Abrir el archivo `index.html` en cualquier navegador web (Chrome, Firefox, Edge, etc.).
